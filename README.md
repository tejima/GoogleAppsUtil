GoogleAppsUtil
===============

GoogleSpreadSeetを活用するためのスクリプト。

▼機能
・一番左の列をステータス行として、カラーリングをしてくれる。

▼コマンド一覧 (この他のコマンド詳細はソースコードを見てください)
A IND RED
B BLUE
C CLEAR 行の色をもとに戻す
D DONE 濃いグレー
E YELLOW
F FORESTGREEN
G GREEN
R RED
L LIGHTGRAY
K 黄緑
P ピンク
BB 空行を手前に一行追加
AA 空行を後に一行追加
X 行を削除
Z スーパーDONE (行を一番下に移動し、グレーにする)
